选择排序与插入排序

May 06, 2018Schlägt

在学习算法中，我们首先学的一般是排序算法，所谓排序就是将一组对象按照某种逻辑顺序重新排列等过程。排序算法中比较经典有：选择排序，插入排序，冒泡排序，哈希排序，快速排序，归并排序等。在这么多排序算法中，选择排序跟插入排序是最基础跟最简单的两种算法。

选择排序

选择排序算法就是，首先找到数组中最大／小的元素，其次，将它和数组中第一个元素交换位置（如果第一个元素就是最小元素那么它和自己交换）。再次，在剩下的元素中找到最小的元素，将它与数组中的第二个元素交换位置。如，如次重复，直到将整个数组排序。这种方法就叫做选择排序。

代码演示

/**
 * 选择排序
 */

private static  void selectionSort(int[] arrays) {


    int length = arrays.length;
    int temp;

    for (int j = 0; j < length; j++) {
        int mindex = j;
        for (int i = j+1; i < length; i++) {
            if (arrays[mindex] > arrays[i]) {
                mindex = i;
            }
        }
        temp = arrays[mindex];
        arrays[mindex] = arrays[j];
        arrays[j] = temp;
    }


}

特点

  1. 运行时间跟输入无关

    这里说的输入无关不是说跟它输入的数据大小无光，而是说跟它输入的数据的特点无关。因为输入的数据无论是否有序或者全部一样它所运行的时间跟无序是一样长的。

  2. 数据移动是最少的。

    每次交换都会改变两个数组的值，因此选择排序用了N次交换，交互次数跟数组的大小是线性关系。相对其他算法的都是线性对数关系或者平方级别关系。

插入排序

插入排序算法就是，从第一个元素开始，然后取出下一个新元素，然后将这个取出来的新元素从左边的序列中，从右往左依次扫描，如果该元素（已排序）大于／小于新元素，将该元素移到下一位置，直到找到已排序的元素小于或者等于新元素的位置，将新元素插入到该位置后。

代码演示：

/**
 * 插入排序
 */
public static void  insertSort(int[] arrays){

    for (int i = 1; i < arrays.length; i++) {
        for (int j = i; j >0; j--) {
            if(arrays[j-1]>arrays[j]){
                swap(arrays,j-1,j);

            }else {
                break;
            }
        }
    }
}

public static void swap(int[] arrays,int i,int j){
   int  temp = arrays[i];
    arrays[i] = arrays[j];
    arrays[j] = temp;
}

特点

插入排序对部分有序的数组十分高效，也很适合小规模大数组
